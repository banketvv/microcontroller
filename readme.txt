##### STM32
1. Для написания кода под плату STM32 требуется установить IDE STM32CubeIDE.
2. Открыть файл с кодом "stm32_code.c" в IDE.
3. Подключить плату к ПК загрузить средствами IDE на неё код. 
4. В результате встроенный светодиод на плате STM32 замигает с интервалом 1 секунда.

3. Подключить плату к ПК загрузить средствами IDE на неё код. 
4. В результате встроенный светодиод на плате ESP32 замигает с интервалом 1 секунда.
